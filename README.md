# ⚖️ Aplicación de Expediente Legal - Abogados Forenses

Esta es una aplicación web profesional diseñada para la **primera entrevista legal** y la generación automatizada de expedientes. Permite a los abogados y clientes recopilar información estructurada de manera intuitiva, colaborativa y eficiente.

## 🚀 Características Principales

### 1. Estructura de Formulario Profesional
La aplicación está dividida en tres secciones clave, diseñadas para cubrir todos los aspectos de un caso legal:
- **I. Datos Generales del Cliente:** Recopilación de identidad, contacto, tipo de asunto y urgencia.
- **II. Información de la Contraparte:** Identificación detallada de la parte contraria, incluyendo domicilio y relación con el cliente.
- **III. Descripción de Documentación:** Inventario de pruebas, análisis preliminar y observaciones especiales.

### 2. Funcionalidades Avanzadas
- **🎤 Dictado por Voz:** Integración con la API de reconocimiento de voz para facilitar el resumen del caso sin necesidad de teclado.
- **➕ Campos Dinámicos:** Capacidad de agregar múltiples bloques de información adicional en cualquiera de las tres secciones.
- **💾 Auto-guardado Local:** El progreso se guarda automáticamente en el navegador cada 30 segundos para evitar pérdida de información.
- **✅ Validación Inteligente:** Asegura que los campos críticos estén completos antes de permitir la exportación.

### 3. Opciones de Exportación Multicanal
- **💬 WhatsApp:** Envío directo del expediente formateado al número configurado (3337 785120).
- **📧 Email:** Generación de un correo electrónico profesional dirigido a `lic.azua@abogadosforenses.org`.
- **🖨️ PDF / Impresión:** Formato optimizado para impresión física o guardado como PDF.
- **📝 Formatos Digitales:** Descarga directa en **Markdown (.md)** y **HTML** con estilos integrados.

## 📋 Instrucciones de Uso

1. **Instalación:** No requiere instalación. Solo asegúrese de que los archivos `index.html`, `style.css` y `script.js` estén en la misma carpeta.
2. **Ejecución:** Abra el archivo `index.html` en cualquier navegador web moderno (Chrome, Edge o Safari recomendados para soporte de voz).
3. **Completado:** Siga las instrucciones en pantalla de cada sección. Use el botón de micrófono para dictar el resumen si lo prefiere.
4. **Exportación:** Una vez finalizado, elija su método de exportación preferido en la parte inferior del formulario.
